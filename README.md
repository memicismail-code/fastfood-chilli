# Fast Food Chilli

Modern web page for **Fast Food Chilli** featuring a premium design, dynamic menu, and WhatsApp-based delivery system.

## Features
- **Modern UI**: Clean, responsive, and vibrant design.
- **Dynamic Menu**: Full list of items including Doners, Burgers, and more.
- **Shopping Cart**: Real-time price calculation and item management.
- **WhatsApp Delivery**: One-click checkout that sends the order details via WhatsApp.
- **Instagram Integration**: Direct links to the official Instagram reels.

## Project Structure
- `index.html`: Main structure and layouts.
- `style.css`: Custom premium styling.
- `app.js`: Application logic and menu data.
